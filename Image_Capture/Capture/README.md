# Image capture Using OpenCV

_We have created a program which will capture an image using webcam everytime we run the program_

**OpenCV is an Image Processing Library**
* Install `OpenCv` library for this program using `pip install opencv-python`
* Run the program
* It will capture the image and store to the given path


**What is OpenCV**

_OpenCV (Open Source Computer Vision Library) is a library of programming functions mainly aimed at real-time computer vision. Originally developed by Intel, it was later supported by Willow Garage then Itseez (which was later acquired by Intel). The library is cross-platform and free for use under the open-source BSD license._

[Source](https://opencv.org/)
